ANL

Introduction to ANL

ANL is a lightweight encoding language created by Aditya Ray. It converts text into a compact code (numbers, tokens and small markers). ANL is implemented in Python and commonly used with an HTML/CSS/JS frontend served by Flask.

ANL — Rules (updated)

  • 1
    Lowercase letters: each becomes its position + . — e.g. a → 1., k → 11.
  • U
    Uppercase letters: encoded with a leading U then the position + . — e.g. A → U1., Z → U26.
  • ␣
    Space: encoded as __ (two underscores).
  • $
    Digits: encoded with a leading $ (e.g. 7 → $7).
  • _
    Special characters: mapped to short underscore tokens (e.g. @ → _1_, + → __1__, . → _20_).
  • .
    Token separator: every letter token ends with a dot (.) so tokens can be parsed easily.

Quick examples

"abcd"      → 1.2.3.4.
"A"         → U1.
"k"         → 11.
"kkasd"     → 11.11.1.19.4.
"Aditya Yadav" → U1.4.9.20.25.1.__U25.1.4.1.22.

Tip: the U prefix removes ambiguity between upper- and lower-case letters so decoding is deterministic.

Technical notes

Use the exact token rules when encoding/decoding. Special tokens are checked longest-first during decoding so tokens like __1__ (double-underscore) are matched before single-underscore tokens.

Example encoded string

U1.4.9.20.25.1.__U25.1.4.1.22._1_  → decodes to "Aditya Yadav@"

Want changes?

If you prefer a different uppercase marker or a different digit prefix, update the encoder/decoder together so they remain compatible.
